Add/Report a Monitoring

To add a new Monitoring record, select the "Report a Monitoring" button from the External Monitorings section of the Home Page. To save your information, select the "Save" button.


Initially, the required data elements are:

  • Project Name
  • Source
  • Notification Date
  • Notification
