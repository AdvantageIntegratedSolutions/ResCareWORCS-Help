Add Participant

To add a new Participant record, select the Participants tab, and then select the "+ New Participant" button. To save your information, select the "Save" button.


The required data elements are:

  • Participant First Name, Participant Last Name, Participant ID
  • Staff Name, Office
  • Participant Address, City, State, and Zip
