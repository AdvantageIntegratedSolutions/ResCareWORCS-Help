Add Funding Change

The Funding Change record is used to store/display information specific to a particular Funding Source's program year budget for a particular service. To add a Funding Change, select the "Add Funding Change" button in the in the Funding Changes section of the Funding Source screen. To save the record, select the "Save" button.


The required data elements are:

  • Funding Source
  • Funds Type
  • Date Effective
  • Expiration Date
  • Status (Defaults to "Approved")
  • Approved By
