Add Case Note

The Case Note record is used to store/display information specific to a particular participant's case. To add a Case Note, select the "Add Case Note" button in the Case Notes section of the Participant screen. To save the record, select the "Save" button.


The required data elements are:

  • Date
  • Type
  • Entered By
  • Comment
